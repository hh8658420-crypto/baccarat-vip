真人荷官视频接入说明：

1. 把你的成年真人女荷官循环视频放到本文件夹，命名为 dealer.mp4。
2. 如有透明背景 WebM，命名为 dealer.webm，网页会优先使用 WebM。
3. 建议视频：竖屏或半身构图，6~15秒循环，静音，720p以内，文件尽量小于 8MB，方便 GitHub Pages 加载。
4. 也可以临时用 URL 参数测试：?v=dealer2&dealer=assets/你的文件.mp4
5. iPhone 自动播放要求视频必须 muted + playsinline，代码已经写好。
6. 没放视频时，会自动显示 poster 兜底图，不影响游戏运行。
